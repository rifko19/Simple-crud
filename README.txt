--------------Jalankan Apache dan MySQL--------------

1. Buka software Xampp, lalu start Apache dan MySQL

2. Pindahkan folder crud ke C:/xampp/htdocs/

3. Pada browser buka url : http://localhost/crud

---------------Membuat database (Xampp)--------------

1. Buat database terlebih dahulu pada myPhpAdmin,
	-Pada browser buka url : localhost
	-Pilih phpMyAdmin pada bagian atas
	-Pada sidebar, pilih new
	-Isi nama database yaitu uas (jangan spasi)
	-sesuaikan jumlah tabel lalu create yaitu ada 6, nama tabel yaitu crud

2. Import
	-Pada bagian database yang telah dibuat tadi pergi kebagian impor
	-pilih file, lalu impor file uas.sql

3. Donee, buka : http://localhost/Welcome.php
